## 一、服务说明

本应用是一款计数器工具应用，主要提供以下功能：

1. **创建和管理计数器**：您可以创建自定义计数器，设置名称、图标、结算周期和计数频次等参数。
2. **计数功能**：对创建的计数器进行计数操作，支持按结算周期自动归档。
3. **历史记录**：查看各结算周期的计数历史记录及详情，支持列表和图表两种展示形式。
4. **提醒功能**：设定提醒时间、频次及触发的地理位置，以便及时提醒您进行计数。
5. **数据管理**：编辑和删除已创建的计数器及其相关数据。

本应用为纯本地工具型应用，所有数据存储在您的设备本地，不涉及用户注册、登录或云端同步服务。
