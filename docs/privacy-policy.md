# 隐私政策

**最后更新日期：2026年3月2日**

**生效日期：2026年3月2日**

我们深知个人信息对您的重要性，我们将按照法律法规要求，采取相应安全保护措施，尽力保护您的个人信息安全。本隐私政策（以下简称"本政策"）适用于 嘀咔（ticca）（以下简称"本应用"）。

请您在使用本应用前，仔细阅读并充分理解本政策。一旦您开始使用本应用，即表示您已阅读并同意本政策。

---

## 一、我们收集的信息

本应用是一款纯本地工具型应用，**不设用户账户系统，不要求注册或登录**。我们致力于最大限度地减少对您个人信息的收集。以下是本应用在运行过程中可能涉及的信息：

### 1.1 您主动提供的信息

- **计数器数据**：您创建的计数器名称、图标设置、结算周期配置、计数频次设置等。
- **计数记录**：您的每次计数操作的时间记录。
- **提醒设置**：您设定的提醒时间和提醒频次。

以上数据均**仅存储在您的设备本地**，我们不会将其上传至任何服务器。

### 1.2 应用运行所需的设备权限及相关信息

- **位置信息**：本应用的提醒功能支持基于地理位置的触发。当您启用此功能时，本应用会请求访问您的设备位置信息，用于判断您是否进入设定的地理位置范围（1千米以内）。位置信息仅在设备本地处理，**不会上传至任何服务器或与第三方共享**。您可以随时在系统设置中关闭位置权限。
- **通知权限**：本应用的提醒功能需要使用系统通知服务。当您启用提醒功能时，本应用会请求通知权限，以向您发送计数提醒。

### 1.3 我们不收集的信息

我们明确声明，本应用**不会**收集以下信息：

- 个人身份信息（姓名、身份证号、电话号码、电子邮箱等）
- 设备唯一标识符（IDFA、IDFV 等）
- 浏览器或搜索历史记录
- 通讯录或联系人信息
- 照片、视频或其他媒体文件
- 健康、健身或医疗数据
- 财务或支付信息
- 任何用于广告追踪的数据

## 二、我们如何使用信息

本应用涉及的所有数据仅用于以下目的：

1. **提供核心功能**：使用您创建的计数器数据和计数记录来实现计数、历史记录查看、图表展示等功能。
2. **提醒服务**：使用您设定的提醒配置和位置信息（如已授权），在符合条件时向您发送提醒通知。
3. **功能改进**：所有数据处理均在设备本地完成，不涉及数据上传或远程分析。

## 三、信息存储与安全

### 3.1 存储方式

本应用的所有数据均存储在您的 iOS 设备本地存储空间中，不会传输至外部服务器。数据的安全性依赖于您的设备安全措施（如设备密码、Face ID/Touch ID 等）。

### 3.2 数据保留

- 您的数据将一直保存在设备上，直到您主动删除计数器数据或卸载本应用。
- 删除计数器时，与该计数器相关的所有数据（包括历史记录）将被永久删除，不可恢复。
- 卸载本应用将导致所有应用数据被永久删除。

### 3.3 安全措施

- 本应用采用 iOS 系统提供的安全存储机制保存数据。
- 由于数据不离开您的设备，因此不存在网络传输过程中的数据泄露风险。

## 四、信息共享与披露

我们**不会**以任何方式向第三方共享、转让或披露您的个人信息，原因如下：

1. 本应用不收集可识别个人身份的信息。
2. 本应用不包含任何第三方分析工具、广告 SDK 或社交分享功能。
3. 本应用不与任何第三方服务进行数据交互。

## 五、第三方服务

本应用**不集成**任何第三方服务，包括但不限于：

- 第三方分析工具（如 Google Analytics、Firebase Analytics 等）
- 广告网络或广告 SDK
- 社交媒体登录或分享 SDK
- 云存储或同步服务
- 崩溃报告工具

## 六、儿童隐私保护

本应用不针对 13 岁以下的儿童，也不会有意收集 13 岁以下儿童的个人信息。由于本应用不收集任何个人身份信息，因此不存在儿童个人信息被收集的情况。如果您是家长或监护人，并认为您的孩子向我们提供了个人信息，请联系我们，我们将采取措施删除相关信息。

## 七、您的权利

根据适用的隐私保护法律法规，您享有以下权利：

1. **访问权**：您可以随时在本应用中查看您创建的所有计数器数据和历史记录。
2. **删除权**：您可以随时删除任何计数器及其相关数据；卸载应用将删除所有数据。
3. **权限管理**：您可以随时在 iOS 系统设置中管理本应用的位置权限和通知权限。
4. **选择权**：位置服务和通知功能均为可选功能，您可以选择不启用，这不会影响本应用核心计数功能的使用。

## 八、隐私政策的变更

我们可能会不时更新本隐私政策。更新后的政策将在本应用内以适当方式通知您，并在本页面上发布修订版本。我们建议您定期查阅本政策以了解最新信息。

如果我们对本政策作出重大变更，我们将在变更生效前通过应用内通知等醒目方式告知您。

## 九、适用法律

本隐私政策的制定、执行和解释均适用中华人民共和国法律（不包括港澳台地区法律），并参照《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》等相关法律法规的要求执行。
