# 点记（ticca）

## 项目简介

轻量计数器工具，自定义周期结算，支持地理围栏提醒，数据纯本地存储，保护您的隐私。

## 应用商店描述

点记（ticca）是一款简洁高效的计数器应用，帮助您轻松记录和管理日常生活中的各类计数需求。

### 核心功能

- **自定义计数器**：自由创建多个计数器，支持设置名称和个性化图标，一目了然。
- **灵活的结算周期**：支持按天、月、年设置结算周期，满足不同场景的统计需求，如月度消费次数、年度目标追踪等。
- **计数频次限制**：可设定单位时间内的计数上限，帮助您合理控制频次。
- **历史记录回顾**：按周期查看历史计数记录，每笔计数时间清晰可查。
- **智能提醒**：支持基于地理位置的围栏提醒，当您到达指定区域时自动触发提醒，不再遗漏。
- **隐私优先**：所有数据仅存储在您的设备本地，无需注册登录，无需联网，您的数据完全由您掌控。

### 适用场景

- 记录每月健身次数
- 追踪饮水、用药等日常习惯
- 统计到店消费次数
- 任何需要周期性计数的场景

点记，让每一次记录都简单而有序。
